Právní asistent ČR

Agent pro M365 Copilot, který vyhledává ve čtyřech právních zdrojích: české zákony (e-Sbírka), judikatura českých soudů, legislativa EU (EUR-Lex) a rozhodnutí Soudního dvora EU. Odpovídá česky, cituje doslovné znění a nikdy si nevymýšlí obsah.

Datové zdroje

  • e-Sbírka zákonů — 46 376 právních předpisů ČR (zákony, vyhlášky, nařízení vlády, nálezy ÚS) od roku 1918, rozčleněných na 112 753 prohledávatelných fragmentů
  • Judikatura českých soudů — rozhodnutí Nejvyššího soudu, Nejvyššího správního soudu, Ústavního soudu a krajských soudů z databáze justice.cz
  • Legislativa EU (EUR-Lex) — nařízení, směrnice a rozhodnutí Evropské unie z databáze EUR-Lex
  • Rozhodnutí SDEU — rozsudky a stanoviska Soudního dvora EU a Tribunálu

Co agent umí

  • Vyhledat v plných textech zákonů, vyhlášek, nařízení vlády a dalších právních předpisů
  • Zobrazit doslovné znění konkrétního paragrafu, článku nebo odstavce
  • Filtrovat podle typu předpisu, roku, účinnosti k datu, sbírky (Sb., Sb.m.s.)
  • Zobrazit metadata předpisu a poslední novely
  • Vyhledat v judikatuře českých soudů — podle soudu, typu rozhodnutí, data, odkazovaného zákona
  • Zobrazit detail rozhodnutí podle ECLI identifikátoru
  • Vyhledat v legislativě EU — nařízení, směrnice, rozhodnutí, filtrace podle roku a typu
  • Zobrazit detail předpisu EU podle CELEX čísla
  • Vyhledat v rozhodnutích SDEU — podle typu řízení, země původu, soudu
  • Kombinovat zdroje — najít český zákon i relevantní EU legislativu k tématu

Co agent neumí / omezení

  • Poskytnout právní radu — agent je vyhledávací nástroj, ne advokát
  • Interpretovat zákon — zobrazuje doslovné znění, interpretaci ponechává na uživateli
  • Vyhledávat v důvodových zprávách k zákonům
  • Sledovat legislativní proces (návrhy zákonů v Parlamentu)

Známé problémy a omezení

Pouze doslovné citace

Agent zobrazuje přesný text zákonů, judikatury a legislativy EU. Neposkytuje právní výklad ani interpretaci. Pro právní posouzení se obraťte na advokáta.

Aktuálnost dat

Data pochází z otevřených zdrojů (e-Sbírka, justice.cz, EUR-Lex). Aktualizace probíhá periodicky — nejnovější novely nebo rozhodnutí mohou být k dispozici s mírným zpožděním.

Struktura dat e-Sbírky

e-Sbírka publikuje jednotlivé předpisy tak, jak byly vyhlášeny (novely, ne konsolidované znění). Dotaz na „§109 zákoníku práce" agent vyřeší přes semantic search — najde příslušné novely, které daný paragraf upravují, a zobrazí jejich doslovné znění.

Dlouhé texty

Některé paragrafy nebo rozhodnutí mají rozsáhlý text. Agent je zobrazí celé (nezestručňuje), proto odpovědi mohou být delší.

Často kladené otázky

Odpovědi na časté dotazy ohledně agenta Právní asistent ČR.

Kde agenta najdu v M365 Copilotu?

Agent je dostupný výhradně v prostředí M365 Copilot. Najdete ho v levém panelu Copilotu, nebo ho zavolejte přes ikonu @ — začněte psát @Právní asistent ČR.

Agent neodpovídá / vrací chybu

Možné příčiny:

  • Backend se probouzí — po delší neaktivitě může server potřebovat pár sekund na start (scale from zero).
  • Dočasný výpadek — Azure služby mohou být krátkodobě nedostupné.

Zkuste dotaz zopakovat za minutu. Pokud problém přetrvává, kontaktujte správce.

Agent si vymýšlí texty zákonů

Agent je nastavený tak, aby nepoužíval vlastní znalosti a čerpal pouze z ověřených zdrojů (e-Sbírka, justice.cz, EUR-Lex). Pokud si přesto vymýšlí:

  • Ověřte, že jste agenta zavolali přes @Právní asistent ČR (ne obecný Copilot)
  • Zkuste dotaz přeformulovat
Jak zadat citaci zákona?

Citaci zadejte ve formátu číslo/rok bez „Sb." — například:

  • 89/2012 — občanský zákoník
  • 262/2006 — zákoník práce
  • 40/2009 — trestní zákoník
  • 500/2004 — správní řád

Můžete také použít název: „zákoník práce", „občanský zákoník", „trestní zákoník" — agent najde citaci sám.

Jak hledat v judikatuře?

Ptejte se přirozeným jazykem, agent automaticky zavolá searchCourtDecisions. Můžete specifikovat:

  • Soud — Nejvyšší soud, Ústavní soud, Nejvyšší správní soud, krajský soud
  • Téma — „judikatura k náhradě škody", „rozhodnutí o neplatnosti výpovědi"
  • ECLI — pokud znáte identifikátor, agent zobrazí plný text
Jak hledat v legislativě EU?

Ptejte se česky, agent prohledá EUR-Lex. Můžete použít:

  • Přirozený jazyk — „Co říká GDPR o právu být zapomenut?"
  • CELEX číslo — „Zobraz detail 32016R0679" (pro GDPR)
  • Typ dokumentu — „Najdi nařízení EU o..."
Jaký je rozdíl mezi searchLaws a getLawSections?

searchLaws prohledává všechny zákony najednou — použijte pro obecné dotazy jako „ochrana spotřebitele" nebo „náhrada škody".

getLawSections zobrazí konkrétní paragraf jednoho zákona — použijte pro „§109 zákoníku práce" nebo „článek 11 Listiny".

Mohu výsledky použít jako právní podklad?

Agent zobrazuje doslovné znění z autentických zdrojů (e-Sbírka, justice.cz, EUR-Lex).

Pro právní posouzení nebo soudní podání se však vždy obraťte na kvalifikovaného právníka. Agent nenahrazuje právní služby.

Co je e-Sbírka?

e-Sbírka zákonů a mezinárodních smluv je úřední publikační platforma Ministerstva vnitra ČR. Obsahuje plné texty všech právních předpisů ČR od roku 1918 do současnosti.

Agent čerpá z otevřených dat e-Sbírky dostupných na www.e-sbirka.cz.

Mohu kombinovat české a EU právo v jednom dotazu?

Ano — agent umí v jednom dotazu prohledat více zdrojů. Například:

  • „Jak je ochrana osobních údajů upravena v českém zákoně i v GDPR?"
  • „Najdi český zákon i EU směrnici o ochraně oznamovatelů"

Agent zavolá příslušné nástroje pro každý zdroj a výsledky zobrazí pohromadě.
