Právní asistent ČR

Agent pro M365 Copilot, který vyhledává v plných textech českých zákonů a právních předpisů z e-Sbírky zákonů a mezinárodních smluv (otevřená data Ministerstva vnitra ČR). Odpovídá česky, cituje doslovné znění zákonů a nikdy si nevymýšlí obsah.

Co agent umí

  • Vyhledat v plných textech zákonů, vyhlášek, nařízení vlády a dalších právních předpisů
  • Zobrazit doslovné znění konkrétního paragrafu, článku nebo odstavce
  • Filtrovat podle typu předpisu (zákon, vyhláška, nařízení vlády, nález ÚS)
  • Filtrovat podle roku, účinnosti k datu, sbírky (Sb., Sb.m.s.)
  • Zobrazit metadata předpisu (název, účinnost, zrušení, oblast úpravy, schvalovatel)
  • Zobrazit křížové odkazy mezi zákony (na co zákon odkazuje a co odkazuje na něj)
  • Zobrazit poslední změny a novely zákonů od zadaného data
  • Zahrnout nebo vyloučit zrušené předpisy
  • Kombinovat filtry — typ předpisu + rok + účinnost + sbírka

Co agent neumí / omezení

  • Poskytnout právní radu — agent je vyhledávací nástroj, ne advokát
  • Interpretovat zákon — zobrazuje doslovné znění, interpretaci ponechává na uživateli
  • Vyhledávat v judikatuře — data neobsahují rozsudky soudů (kromě nálezů ÚS)
  • Vyhledávat v důvodových zprávách k zákonům
  • Sledovat legislativní proces (návrhy zákonů v Parlamentu)
  • Odpovídat na dotazy mimo české právo

Známé problémy a omezení

Pouze doslovné citace

Agent zobrazuje přesný text zákonů z e-Sbírky. Neposkytuje právní výklad ani interpretaci. Pro právní posouzení se obraťte na advokáta.

Bez judikatury

Data neobsahují rozsudky soudů — agent pracuje pouze s právními předpisy (zákony, vyhlášky, nařízení vlády, nálezy ÚS). Judikaturu hledejte v ASPI, Beck-online nebo na justice.cz.

Aktuálnost dat

Data pochází z otevřených dat e-Sbírky (Ministerstvo vnitra ČR). Aktualizace probíhá periodicky — nejnovější novely mohou být k dispozici s mírným zpožděním.

Struktura dat e-Sbírky

e-Sbírka publikuje jednotlivé předpisy tak, jak byly vyhlášeny (novely, ne konsolidované znění). Dotaz na „§109 zákoníku práce" agent vyřeší přes semantic search — najde příslušné novely, které daný paragraf upravují, a zobrazí jejich doslovné znění.

Dlouhé paragrafy

Některé paragrafy mají rozsáhlý text. Agent je zobrazí celé (nezestručňuje), proto odpovědi mohou být delší.

Často kladené otázky

Odpovědi na časté dotazy ohledně agenta Právní asistent ČR.

Kde agenta najdu v M365 Copilotu?

Agent je dostupný výhradně v prostředí M365 Copilot. Najdete ho v levém panelu Copilotu, nebo ho zavolejte přes ikonu @ — začněte psát @Právní asistent ČR.

Agent neodpovídá / vrací chybu

Možné příčiny:

  • Backend se probouzí — po delší neaktivitě může server potřebovat pár sekund na start (scale from zero).
  • Dočasný výpadek — Azure služby mohou být krátkodobě nedostupné.

Zkuste dotaz zopakovat za minutu. Pokud problém přetrvává, kontaktujte správce.

Agent si vymýšlí texty zákonů

Agent je nastavený tak, aby nepoužíval vlastní znalosti a čerpal pouze z e-Sbírky. Pokud si přesto vymýšlí:

  • Ověřte, že jste agenta zavolali přes @Právní asistent ČR (ne obecný Copilot)
  • Zkuste dotaz přeformulovat
Jak zadat citaci zákona?

Citaci zadejte ve formátu číslo/rok bez „Sb." — například:

  • 89/2012 — občanský zákoník
  • 262/2006 — zákoník práce
  • 40/2009 — trestní zákoník
  • 500/2004 — správní řád

Můžete také použít název: „zákoník práce", „občanský zákoník", „trestní zákoník" — agent najde citaci sám.

Jaký je rozdíl mezi searchLaws a getLawSections?

searchLaws prohledává všechny zákony najednou — použijte pro obecné dotazy jako „ochrana spotřebitele" nebo „náhrada škody".

getLawSections zobrazí konkrétní paragraf jednoho zákona — použijte pro „§109 zákoníku práce" nebo „článek 11 Listiny".

Mohu výsledky použít jako právní podklad?

Agent zobrazuje doslovné znění zákonů z e-Sbírky. Texty jsou autentické, protože pochází z úředního zdroje (Ministerstvo vnitra ČR).

Pro právní posouzení nebo soudní podání se však vždy obraťte na kvalifikovaného právníka. Agent nenahrazuje právní služby.

Co je e-Sbírka?

e-Sbírka zákonů a mezinárodních smluv je úřední publikační platforma Ministerstva vnitra ČR. Obsahuje plné texty všech právních předpisů ČR od roku 1918 do současnosti.

Agent čerpá z otevřených dat e-Sbírky dostupných na www.e-sbirka.cz.

Proč agent nezná judikaturu?

Agent pracuje s právními předpisy (zákony, vyhlášky, nařízení vlády, nálezy ÚS). Rozsudky obecných soudů jsou spravovány Ministerstvem spravedlnosti a nejsou součástí e-Sbírky.

Pro judikaturu použijte justice.cz, ASPI nebo Beck-online.

Proč agent u některých zákonů nevrátí konkrétní paragraf?

e-Sbírka publikuje právní předpisy jednotlivě, jak byly vyhlášeny — tj. novely, ne konsolidované texty. Proto dotaz „§109 zákoníku práce (262/2006 Sb.)" nemusí najít přímý záznam pod touto citací.

Agent v takovém případě automaticky použije sémantické vyhledávání a najde příslušné novely, jejichž text daný paragraf upravuje. Výsledek je stejný — doslovné znění z e-Sbírky.
